# SwagLabAutomation

Automation testing framework for [SauceDemo](https://www.saucedemo.com) 
built using Java, Selenium WebDriver, Cucumber BDD, TestNG and Maven.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Cucumber BDD
- Maven
- Page Object Model (POM)
- Allure Reports

## How to Run
git clone https://github.com/RBHAGAT2604/SwagLabAutomation.git
mvn clean test

## Test Coverage
- Login (valid/invalid scenarios)
- Product browsing & sorting
- Cart management
- Checkout workflow
- Cross-browser testing (Chrome, Firefox, Edge)
